# 🚀 IGNITE 2026 - Official Event Portal

**Ignite 2026** is a high-performance, responsive web application built for the annual tech-fest at **CGPIT**. It features a cyberpunk-inspired aesthetic with real-time interactivity.

## ✨ Key Features
* **Live Countdown**: A real-time synchronization clock targeting the launch on **March 27, 2026**.
* **Interactive Event Archive**: Dynamic search and filtering system for Technical, E-sports, and Non-tech events.
* **Tabbed Schedule**: A clean, categorized timeline for Day 1, Day 2, and Day 3 events.
* **System-Alert Toasts**: Real-time "System Online" notifications for user engagement.
* **Responsive UI**: Fully optimized for mobile, tablet, and desktop using **Tailwind CSS**.

## 🛠️ Tech Stack
* **Frontend**: HTML5, Tailwind CSS (Utility-first styling).
* **Logic**: Vanilla JavaScript (DOM Manipulation, Date/Time API).
* **Deployment**: Hosted on **Render** with CI/CD integration via GitHub.
